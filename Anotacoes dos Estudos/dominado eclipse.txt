ALT + SHIFT + M - COPIA O CÓDIDO SELECIONADO PARA DENTRO DE UM MÉTODO - REFATORAÇÃO DA CLASSE.
ALT + SETA PARA BAIXO OU PARA CIMA LEVA A LINHA PARA QUALQUER POSIÇÃO DA CLASSE.
CTRL + 3 + GGAS - GERA OS MÉTODOS GETS E SETS
CTRL + 3 + GCUF - GERA O CONTRUTOR COM FILDS
CTRL + SHIFT + R - BUSCA QUALQUER ARQUIVO PELO NOME
CTRL + SHIFT + T - BUSCA QUALQUER CLASSE JAVA PELO NOME
CTRL + O - LISTA AS VARIAVEIS E MÉTODOS QUE ESTÃO DENTRO DA CLASSE
CTRL + T - AO POSICIONAR O CURSO SOBRE UMA INTERFACE E USAR O ATALHO É LISTADO TODAS AS CLASSES QUE A IMPLEMENTA, SE FOR ENCIMA DE UMA CLASSE LISTA AS QUE A EXTENDE
ALT + SETA PARA A DIREITA OU ESQUESTA - NAVEGA PARA AS TELA ONDE VISITADA ALTERIORMENTE, CASO JÁ ESTIVER FECHADA VÃO ABRINDO NOVAMENTE
CTRL + PAGE UP OU PAGE DOWN - NAVEGA ENTRE AS PAGINAS ABERTAS.
CTRL + ALT + H - AO CLICAR NO MÉTODO E USAR O ATALHO É MOSTRATO ONDE O MÉTODO ESTA SENDO USADO, MOSTRA A HIERARQUIA DAS CLASSES ATÉ CHEGAR NO MÉTODO.
ALT + SHIFT + R - RENOMEA  A CLASSE, VARIAVEL OU MÉTODO EM TODAS AS REFERÊNCIAS.
CTRL + SHIFT + W - FECHA TODAS AS JANELAS ABERTAS.
CTRL + SHIFT + L - LISTA TODAS AS TECLAS DE ATALHO.

*************DEBUG**********************
CTRL + R - QUANDO ESTIVER DEBUGANDO, O DEBUG PAROU EM UMA LINHA, É SÓ COLOCAR O MOUSER NO PROXIMO LIGAR E USAR O ATALHO QUE O DEBUG PARA LÁ.
CTRL + SHIFT + D - CASO QUEIRA VISUALIZAR O VALOR DE UMA VARIAVEL QUE RETORNA DE UM METODO POR EXEMPLO É SÓ SELECIONAR A LINHA EM QUE PAROU O DEBUG E USAR O ATALHO,
					USANDO ESSE ATALHO 2 VEZES SURGE A TELA DISPLAY ON PODE EDITAR O VALOR DAS VARIAVEIS ESTA ESTA PASSANDO PARA O METODO E JÁ OBTER O RESULTADO.
F7 - QUANDO NO DEBUG ENTRA DENTRO DE UMA CLASSE PARA EXECUTAR UM MÉTODO, USA ESSE ATALHO PARA RETORNAR PARA ONDE CHAMOU O METODO.
F5 - ENTRA NO MÉTODO.
F6 - PROXIMA LINHA.
